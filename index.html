Once Firebase is set up, any change you or Sarah makes will appear on each other's device within seconds.
📋 5-minute Firebase setup:
1. Go to console.firebase.google.com → Create a project (call it rathbun-family-hub)
2. Click Realtime Database → Create database → Start in test mode
3. Go to Project Settings ⚙️ → Your apps → Web → Register app → Copy the config
4. Open github.com/rathbunb/family-hub → Click index.html → ✏️ Edit
5. Find FIREBASE_CONFIG = { and paste your values → Commit changes
6. Wait 60 seconds → refresh the app → badge turns green ✓
